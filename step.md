# STEP 0

install typescript
```
npm i -g typescript
```

initial project

```
npm init -y
```

install fastify

```
npm i fastify -S
```

install typings
```
npm i -D typescript @types/node
```

initial typescript project
```
tsc --init 
```

create `src` folder

create `typings` folder

edit `tsconfig.json`

create `.gitignore`

create `server.ts`

edit `pacakge.json` for start `nodemone` with command `npm start`

start server with command `npm start`

open `http://127.0.0.1:8080`
